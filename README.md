Repository Contents
Data Files
Firefly.xlsx - Primary specimen collection data containing:
Taxonomic information (Genus, Species)
Geographic data (Country, Province/Department/State, Location, Latitude, Longitude)
Collection metadata (Date, Repository information)
Additional specimen identifiers and labels

Measurements.xlsx - Species-level morphometric data containing:
Species names
Average body length (mm)
Average body width (mm)

Code Files
firefly_analysis.Rmd - Main R Markdown document containing all data cleaning, analysis, and visualization code. This file generates the final HTML report.

Output Files
firefly_analysis.html - Rendered HTML report with figures, tables, and narrative results

Data Sources and Citations
